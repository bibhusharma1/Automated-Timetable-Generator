# Automated-Timetable-Generator
An automated timetable generator that applies logical constraints and optimization techniques to generate efficient, clash-free academic schedules, reducing manual effort and human error.
# Clear Problem Definition
The project was initiated by formally defining the problem of manual academic timetable creation, which is time_consuming, error_prone and difficult to scsle wth increasing constraints such as faculty availability ,classroom capacity and subject distribution.
